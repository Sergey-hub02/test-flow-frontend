# Клиентская часть приложения "Test Flow"
Система TestFlow является информационной системой тестирования учащихся, использующая методы нелинейного оценивания для повышения достоверности оценки успеваемости.

Проект разработан в рамках магистерской диссертации студента 2 курса РТУ МИРЭА Пака С.А. Проект носит учебно-демонстрационный характер
